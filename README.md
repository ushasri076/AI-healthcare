# AI Healthcare Assistant 🏥

## Overview
This project is an AI-powered hospital receptionist prototype that helps patients interact through a chatbot, book appointments, and get health-related assistance.

## Features
- Chatbot interaction
- Appointment booking
- Patient details collection
- Health tips
- Medicine purchase interface

## Tech Used
- HTML, CSS

## Future Scope
- React + Tailwind frontend
- FastAPI backend
- LangGraph AI workflow
- Supabase database
